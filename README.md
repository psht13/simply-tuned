# SimplyTuned

A minimal iOS (latest-only) guitar tuner.

## Run on device
1. Open `simply-tuned.xcodeproj` in Xcode.
2. Select a physical iPhone (not a simulator).
3. Configure signing for the `simply-tuned` target if needed.
4. Build & Run.

## Microphone permission
Microphone access is required for pitch detection; iOS will prompt on first launch.
